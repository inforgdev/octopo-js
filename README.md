## About

OctopoJS is a CSS preprocessor AST stringifier.

## Installation

Download and install via NPM from Github repository:

```bash
$ npm install --save-dev inforgdev/octopo-js
```

For usage using ESM, import it in JS file:

```js
import "octopo-js";
```
